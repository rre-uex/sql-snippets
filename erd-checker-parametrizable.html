ERD Solution Checker (Parametrizable)

Enter your ERD solution in the text area below using bigER notation.

How to use this checker:

This page expects the expected solution to be provided as a base64-encoded URL parameter.

URL format:

erd-checker-parametrizable.html?solution=BASE64_ENCODED_ERD

Example JavaScript to encode:

const solution = "your ERD solution here";
const encoded = btoa(encodeURIComponent(solution));
const url = `erd-checker-parametrizable.html?solution=${encoded}`;
